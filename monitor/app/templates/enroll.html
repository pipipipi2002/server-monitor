Uninstall an agent

Run on the host you want to decommission. Stops the service, removes the binary, the agent token, and (on Windows) the monitor's CA cert from the trust store. The server's row stays in this table afterwards with an agent offline badge — remove it manually via the SQLite recipe in the README when you're done.

Linux — run as root

For host with CA root installed:

curl -fsSL https://{{ monitor_host }}/uninstall.sh | sudo bash

For host with no CA root installed (e.g. Traefik HTTP path):

curl -fsSL http://{{ monitor_host }}/uninstall.sh | sudo bash
Windows — run from elevated PowerShell

For host with CA root installed:

iwr https://{{ monitor_host }}/uninstall.ps1 -UseBasicParsing | iex
Uninstall-MonitorAgent

For host with no CA root installed (e.g. Traefik HTTP path):

iwr http://{{ monitor_host }}/uninstall.ps1 -UseBasicParsing | iex
Uninstall-MonitorAgent
